In order to make a core data application do the following.

1. Create a new project using "Single View Application".
2. Select "Use CoreData".
3. Select a file that ends with ".xcdatamodelId"
4. Create your entity.
5. Add attributes.
6. With the entity selected, hover over "Editor" in top menu.
7. Click "Create NSManagedObject Subclass".
8. Now you can build your UI and follow what my code has.
